9. 코드 블록 복사 버튼 (이미 커밋됨)

신호등 우측에 복사 버튼이 생깁니다.

ts 
export async function fetchPost(id: string) {
  const res = await fetch(`/api/blogs/${id}`);
  return res.json();
}
이 프리뷰는 정적 파일입니다. 실제 구현 전에 디자인 방향을 합의하기 위한 목적이며, 선택된 항목만 골라서 반영하면 됩니다.
